## Structure

The website is organized into main categories accessible via:

1. **Sidebar Navigation** (Left)
   - Hierarchical topic organization
   - Expandable/collapsible sections
   - Search box for quick access

2. **Top Navigation Bar**
   - Quran - Direct access to Quranic topics
   - Hadith - Hadith sciences and collections
   - Glossary - Islamic terminology
   - Blog - Articles and contemporary topics

3. **Main Content Area**
   - Detailed explanations
   - Quranic verses and Hadith references
   - Beautiful formatting with quote boxes
   - Arabic text where appropriate

## Customization

The website uses inline CSS and JavaScript, making it easy to customize:

- **Colors**: Modify the gradient values in the CSS
- **Content**: Update the `contentDatabase` object in the JavaScript
- **Navigation**: Edit the sidebar HTML structure
- **Styling**: All styles are in the `<style>` section
